Example
Description
AuthLock
rsslAuthLock demonstrates DACS library functionality, including:
•
DACS_GetLock(): Creates a DACS access lock from a list of entitlement codes (i.e., codes specified by the data vendor on which permissioning is based).
•
DACS_CmpLock(): Compares two access locks for equality.
•
DACS_CsLock(): Combines a list of access locks into a single composite access DACS lock.
•
DACS_perror(): Creates a textual message describing the last error generated by a DACS library call.
