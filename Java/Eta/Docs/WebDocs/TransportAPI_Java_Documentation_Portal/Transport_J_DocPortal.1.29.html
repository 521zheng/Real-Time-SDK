Note: Transport API Java example programs are not intended for use in production environments nor to measure performance. A separate, performance test tool package is available for evaluation of the performance of the Transport API. The performance test package consists of performance test programs, configuration files tuned for high throughput or low latency, and a document containing the results of performance tests, including information about the machines used in the tests. This can serve as a guideline to fine tune your own Transport API applications to achieve similar or better results.
For more information on this performance package, contact your Thomson Reuters representative.
