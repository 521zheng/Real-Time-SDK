Transport API Java Edition

Running Examples
You can run most Transport API examples directly from a Command Line prompt. All example configuration and parameters are provided as command line arguments (the -? argument displays a list of all configuration parameters). All parameters have default values to allow basic functionality when running providers and consumers on the same machine.
For example, to run the Consumer example using its default values, use the following command line:
 
 
Consumer
 
or specify any of the following options on the command line:
 
 
Consumer [-uname LoginUsername] [-h ServerHostname] [-p ServerPortNo] [-bc hostname:port]
        [-c hostname:port servicename domain:item_name,...] [-connectionType connectionType]
        [-s ServiceName] [-view] [-post] [-offpost] [-snapshot] [-sl [SymbolList_Name]]
        [-mp|-mpps MarketPrice_ItemName] [-mbo|-mbops MarketByOrder_ItemName]
        [-mbp|-mbpps MarketByPrice_ItemName] [-yc|-ycps YieldCurve_ItemName] [-runtime seconds]
        [-proxy host:port] [-x] [-publisherInfo]
