Transport API Java Edition

PDFs and Reference Manuals
•
Transport API Java Edition Developers Guide
The Transport API Developers Guide includes information and examples that aid in developing with the Transport API product. It provides detailed descriptions for OMM Codec package concepts and use of the Transport package.
•
Transport API Java Edition RDM Usage Guide
The Transport API Java Edition RDM Usage Guide describes the content layout and use for Thomson Reuters-provided domain message models. The guide describes content available through the Thomson Reuters Enterprise Platform, EleKtron, Reuters Data Feed Direct, and various other sources.
•
Transport API Java Reference Guide
This is a Javadoc-generated reference guide, based on the comments and usage information provided in the Transport API Java code files. The Transport API Reference supplements the Transport API Java Developers Guide.
•
Transport API Java Edition Value Added Components Developers Guide
This document provides information and examples that aid you in developing with Transport API Java Edition Value Added Components. This includes information about the Transport API Reactor and Domain Representation helpers.
•
Transport API Java Value Added Components Reference Guide
This is a Javadoc-generated reference guide, based on the comments and usage information provided in the Transport API Java Value Added Components code files. The Transport API Value Added Components Reference supplements the Transport API Java Value Added Components Developers Guide.
•
Transport API Java Edition Open Source Performance Tools Guide
This guide documents the general design and usage of the tools provided for measuring the performance of the Java Transport API. It describes how features of the API send and receive data with high throughput and low latency. This information applies both when the API is directly connected to itself as well as through various Thomson Reuters Enterprise Platform (TREP) components (i.e., the Advanced Data Hub (ADH) and Advanced Distribution Server (ADS)).
•
DACS Lock Developers Guide
This document describes DACS Lock creation and manipulation of permission information. The DACS Lock package is made available as a supplement to the Java Transport API so applications can properly create any necessary permission data for content they provide.
•
ANSI Page Developers Guide
This document describes ANSI Page encoding and decoding. The ANSI package is made available as a supplement to the Java Transport API so, if needed, an application can provide or consume ANSI Page based content.
