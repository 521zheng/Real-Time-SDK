Value Added Components Functionality
Value Added components simplify usage and reduce the amount of code needed to create OMM-based applications. In addition to previously mentioned Transport API features, these components provide the following features:
•
The Reactor component is a connection management and event processing component.
•
For OMM Consumers, the Reactor optionally features an internal, per-channel, watchlist which stores requests opened by the application. This watchlist allows the Reactor to perform actions that simplify application code, such as the recovery of item streams and fan out of group status.
•
The Administration Domain Model Representations reduce the amount of code required to interact with the RDM administrative domains.
•
The Value Added Utilities are a collection of helper constructs mainly used by the Transport API Reactor.
