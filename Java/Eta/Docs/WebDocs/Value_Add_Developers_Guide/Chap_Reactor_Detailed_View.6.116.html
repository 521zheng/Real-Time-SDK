Tunnel Stream Submit
The submit method is used to write content to the TunnelStream. This method also enforces any specified behaviors on submitted content (e.g., if guaranteed messaging is specified, this content follows all configured persistence options).
 
Table 72: Tunnel Stream Submit Method  
METHOD NAME
DESCRIPTION
submit
Use the submit method to pass in opaque or RDM Message content (including Queue Messages) to be processed and sent over the TunnelStream.
This method has additional options that can be specified via TunnelStreamSubmitOptions (refer to Section 6.7.8.3).
