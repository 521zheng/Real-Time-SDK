Generic XML Schema for Directory Entry
The top-level XML schema for DirectoryGroup is as follows:
 
 
<DirectoryGroup>
    <DefaultDirectory value="…"/>
        <DirectoryList>
            <Directory>
                <Name value="…"/>
                <Service>
                    <Name value="…"/>
                    <InfoFilter>
                        ...
                    </InfoFilter>
                    <StateFilter>
                        …
                    </StateFilter>
                </Service>
            ...
        </Directory>
        ...
    </DirectoryList>
</DirectoryGroup>
