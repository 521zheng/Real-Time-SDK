Decoding Field List Contents with Field and Enumerated Types Dictionaries
By itself, a FieldEntry contains only the FieldId and its associated encoded value in Data. Because the Elektron Message API internally stores pre-decoded data, an application can easily decode a FieldEntry (without cross-referencing the FieldId to the correct Field Dictionary to determine its type).
