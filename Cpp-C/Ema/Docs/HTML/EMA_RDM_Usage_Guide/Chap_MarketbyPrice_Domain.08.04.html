Market By Price Update Message
A Market By Price update message is encoded using UpdateMsg and sent by OMM interactive provider and OMM non-interactive provider applications. The provider can send an update message to add, update, or remove price point information. Updates will not be received before images. True snapshots are supported.
 
Note: The provider should send Name and ServiceName only in the first Refresh response message. However if MsgKeyInUpdates is set to true, then Name and ServiceName must be provided for every Update response message.
