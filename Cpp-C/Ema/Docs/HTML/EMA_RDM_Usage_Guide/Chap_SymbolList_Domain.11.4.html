Symbol List Update Message
A Symbol List Update Message is encoded using UpdateMsg and sent by OMM provider and OMM non-interactive provider applications. It adds or removes items from the list. Updates will not be received before images, and a true snapshot is supported.
 
Note: The provider should send the Name and ServiceName only in the first Refresh response message. However if MsgKeyInUpdates is set to true, then the Name and ServiceName must be provided for every Update response message.
