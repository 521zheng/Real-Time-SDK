Element List Contents
To encode a Symbol List request message that specifies interest in data, include an element list with the following element entry:
 
Table 75: Request Message Payload for Symbol List Domain Specifying Symbol List Behaviors  
ELEMENT NAME
TYPE
Default
DESCRIPTION
:SymbolListBehaviors
ElementList
ElementList containing ElementEntry of DataStreams set to 0.
Indicates any expected data behavior of individual items that will be opened from the symbol list.
If this element is absent, individual streams will not be opened.
The contents of :SymbolListBehaviors are extensible.
