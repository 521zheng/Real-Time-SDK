Overview
The Elektron Message API (EMA) targets access to Thomson Reuters Elektron, Thomson Reuters Enterprise Platform (TREP) and Thomson Reuters Direct Feed via an ease-of-use solution. EMA is ideal for users who need light weight Open Message Model (OMM) access with minimal lines of code.
This section provides a very brief overview of EMA and complements the accompanied interface and example usage. Presently, EMA solely defines an abridged interface to solicit feedback. Intent is for Thomson Reuters to release an interface and library implementation in the future.
Readers of this content should be moderately familiar with Elektron, TREP, Direct Feed, and OMM APIs.
