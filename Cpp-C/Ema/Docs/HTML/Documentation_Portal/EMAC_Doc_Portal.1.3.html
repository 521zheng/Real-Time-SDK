API Concepts Guide is written by Elektron API programmers and architects. This guide documents the functionality and capabilities of the Elektron APIs. In addition to connecting to itself, an Elektron API can also connect to and leverage many different Thomson Reuters and customer components. If you want an Elektron API to interact with other components, consult that specific component’s documentation to determine the best way to configure and interact with these other devices.
