FLAG ENUMERATION
MEANING
RSSL_ELF_HAS_ELEMENT_LIST_INFO
Indicates the presence of the elementListNum member. This member is provided as part of the initial refresh message on a stream or on the first refresh message after a CLEAR_CACHE command.
RSSL_ELF_HAS_STANDARD_DATA
Indicates that the RsslElementList contains standard element name, dataType, value-encoded data. You can set this value in addition to RSSL_ELF_HAS_SET_DATA if both standard and set-defined data are present in this RsslElementList. If the RsslElementList does not have entries, do not set this flag value.
