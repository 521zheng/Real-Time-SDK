RSSL Generic Message Utility Function
The Transport API provides the following utility function for use with RsslGenericMsg.
 
Table 177: RsslGenericMsg Utility Functions  
FUNCTION NAME
DESCRIPTION
rsslClearGenericMsg
Clears an RsslGenericMsg structure. Useful for structure reuse.
rsslSetGenericCompleteFlag
Sets the RSSL_GNMF_MESSAGE_COMPLETE flag on an already encoded
RsslGenericMsg.
rsslUnsetGenericCompleteFlag
Removes the RSSL_GNMF_MESSAGE_COMPLETE flag on an already encoded
RsslGenericMsg.
