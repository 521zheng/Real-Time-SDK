VAConsumer
rsslVAConsumer is an example of an OMM consumer written with the ValueAdd library and demonstrates how to consume data from an OMM Provider using Value Added Components.
Click here to access the readme.
VANIProvider
rsslVANIProvider is an example of an OMM non-interactive provider written with the ValueAdd Library and demonstrates how to non-interactively provide Level I Market Price data to an Advanced Data Hub (ADH) using Value Added Components.
Click here to access the readme.
VAProvider
rsslVAProvider is an example of an OMM interactive provider written with the ValueAdd Library and demonstrates how to interactively provide data to OMM consumers using Value Added components.
Click here to access the readme.
rsslWatchlistConsumer
rsslWatchlistConsumer is an example of an OMM consumer written with the ValueAdd library and demonstrates how to consume data from an OMM provider that uses the built-in watchlist feature of the Value Added Components. When enabled, various behaviors of recovery and aggregation are performed that you would otherwise need to implement yourself.
Click here to access the readme.
