Developer Community Portal
The Developer Community is a Thomson Reuters portal designed for an engaging and educating developer experience. Using the portal, developers can download the latest packages and documentation sets, access video tutorials and examples, obtain quick start materials, and interact with other API developers, including ones who worked on the actual API code. The portal also includes an actively managed Question and Answer forum. Anyone can access the Developer Community Portal, though to do so you must first create an account.
To access product support, communities, and to contact Thomson Reuters, visit us on the following websites:
•
For queries: https://customers.reuters.com/developer/crmcontactus/support.asp
•
To access the Developer Community Portal: https://developers.thomsonreuters.com/
•
For the latest API downloads: https://developers.thomsonreuters.com/elektron/elektron-sdk-cc/downloads
•
For API tutorials: https://developers.thomsonreuters.com/elektron/elektron-sdk-cc/learning
•
Developer’s Q&A forum: https://developers.thomsonreuters.com/elektron/elektron-sdk-cc/qa
Additionally, Transport API training is available in classroom settings, self-paced eLearning packages, and through free webcast eLearning modules. For details, contact your account manager.
