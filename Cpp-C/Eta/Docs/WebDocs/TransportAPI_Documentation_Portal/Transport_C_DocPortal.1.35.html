Description
AuthLock
rsslAuthLock demonstrates DACS library functionality, including:
•
DACS_GetLock(): Creates a DACS lock from a list of entitlement codes (i.e., codes specified by the data vendor on which permissioning is based).
•
DACS_CmpLock(): Compares two locks for equality.
•
DACS_CsLock(): Combines a list of locks into a single composite DACS lock.
•
DACS_perror(): Creates a textual message describing the last error generated by a DACS library call.
