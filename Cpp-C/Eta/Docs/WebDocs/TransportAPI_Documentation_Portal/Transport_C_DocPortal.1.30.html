Running Examples
You can run most Transport API examples directly from a Command Line prompt. All example configuration and parameters are provided as command line arguments (the -? argument displays a list of all configuration parameters). All parameters have default values to allow basic functionality when running providers and consumers on the same machine.
For example, to run the rsslConsumer example using its default values, use the following command line:
 
 
rsslConsumer
 
or specify any of the following options on the command line:
 
 
rsslConsumer [-uname LoginUsername] [-h ServerHostname] [-p ServerPortNo] [-c ConnectionType]
        [-s ServiceName] [-view] [-post] [-offpost] [-snapshot] [-sl [SymbolListName]]
        [-mp|-mpps MarketPriceItemName] [-mbo|-mbops MarketByOrderItemName]
        [-mbp|-mbpps MarketByPriceItemName] [-yc|-ycps YieldCurveItemName] [-runtime seconds]
