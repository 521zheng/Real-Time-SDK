Transport API C Edition

Administrative Domain Model Representations
Administrative domain model representations:
•
Are RDM-specific.
•
Provide structures that represent specific messages defined in the Transport API C Edition RDM Usage Guide for the Login, Source Directory, and Dictionary domains.
•
Handle the encoding and decoding of OMM Administrative domain messages, significantly reducing the amount of code necessary to interact with those domain models.
See also Administrative Domain Model Representations Reference Manual.
