Undefined Color Default Value
The default color and the color of erased characters is not defined by the ANSI standard. Thomson Reuters Enterprise Platform applications normally display this as black. It is represented in the data structure by the special value 0x0F. The default monochrome color follows:
 
 
/*define for no colors set*/
#define MON       0xff /* monochrome */
