Yield Curve Update Message
A Yield Curve Update Message is sent by OMM provider and OMM non-interactive provider applications. It conveys any changes to an itemâ€™s data.
 
Warning! Although the payload is an RsslFieldList, some field entries are sent as more complex types such as RsslVector and RsslArray. Encoding and decoding applications should be aware of this and ensure proper handling of these types.
