Source Directory Refresh and Update Payload
A list of services is represented by an RsslMap. Each RsslMapEntry represents a known service and is uniquely identified by its msgKey.serviceId (i.e., its key).
The information about each service is represented as an RsslFilterList. Each RsslFilterEntry contains one of six different categories of information. These categories should correspond to the msgKey.filter member of the refresh or update.
For example layouts of these messages, refer to Section 4.5.
