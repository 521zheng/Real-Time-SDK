Dictionary Refresh Message
A Dictionary refresh message is encoded using and sent by OMM Interactive and non-interactive provider applications and provides the consumer with the content of the requested dictionary.
Since the refresh messages for Dictionary data might be in multiple response messages, the provider should send only and ServiceName in the first refresh message. If MsgKeyInUpdates is set to true, and ServiceName must be provided for all refresh and status messages.
