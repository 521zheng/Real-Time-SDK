Removing Reactor Channels
rsslReactorClose Function
You use the following function to remove an RsslReactorChannel from an RsslReactor instance. It can also close and clean up resources associated with the RsslReactorChannel.
 
Table 29: rsslReactorCloseChannel Function  
FUNCTION NAME
DESCRIPTION
rsslReactorCloseChannel
Removes an RsslReactorChannel from the corresponding RsslReactor and cleans up associated resources. This additionally invokes the rsslCloseChannel function, as described in the Transport API C Edition Developers Guide, to clean up any resources associated with the underlying RsslChannel.
This function can be called from either outside or within a callback.
