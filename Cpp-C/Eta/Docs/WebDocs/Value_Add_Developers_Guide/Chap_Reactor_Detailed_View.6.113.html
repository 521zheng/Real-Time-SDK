RsslReactorRejectTunnelStreamOptions
 
Table 73: RsslReactorRejectTunnelStreamOptions Options  
Option
DESCRIPTION
state
An RsslState to send to the consumer. The application can use the state.streamState, state.dataState, and state.text to indicate the nature of the rejection.
pCos
An optional RsslClassOfService to send to the consumer. If rejecting the stream due to a problem with the RsslClassOfService parameters from the RsslTunnelStreamRequestEvent, the provider application should populate this with the associated parameters.
