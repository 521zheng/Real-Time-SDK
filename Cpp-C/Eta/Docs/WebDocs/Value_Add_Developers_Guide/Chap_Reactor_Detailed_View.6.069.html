Reactor RDM Dictionary Message Event
The RsslRDMDictionaryMsgEvent is returned to the application via the RsslRDMDictionaryMsgCallback.
 
Table 43: RsslRDMDictionaryMsgEvent Structure Members  
Structure MEMBER
DESCRIPTION
baseMsgEvent
An RsslMsgEvent populated with the raw buffer (RsslMsg) and any error information. This structure is defined in Section 6.5.4.1.
pRDMDictionaryMsg
The RDM representation of the decoded Dictionary message. If not present, an error was encountered while processing the information.
This message is presented as the RsslRDMDictionaryMsg, described in Section 7.5.
