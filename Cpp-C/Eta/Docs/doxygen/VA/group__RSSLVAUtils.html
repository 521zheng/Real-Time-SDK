static RsslRDMMsg* rsslCreateRDMMsgCopy ( RsslRDMMsg *  pRdmMsg,
RsslUInt32  lengthHint,
RsslRet *  pRet 
) [static]
Parameters:
pRdmMsgRsslRDMMsg to copy.
lengthHintAn initial size for the memory buffer created. If this is too small a larger size will be allocated.
Returns:
The created RDMMsg, or NULL if an error occurred. If so, pRet is populated with:
  • RSSL_RET_INVALID_ARGUMENT, if the message appears invalid.
  • RSSL_RET_BUFFER_NO_BUFFERS, if memory could not be allocated.
  • RSSL_RET_FAILURE, if the message could not be copied for some other reason.
