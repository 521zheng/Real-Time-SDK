RSSL RDM Login Codec
RSSL RDM Codec API

The RSSL RDM Login component is intended to assist developers with encoding and decoding messages used by the Login Reuters Domain Model. This component provides typed data structures, encoding and decoding functions for Login Domain messages.

union  RsslRDMLoginMsg
 The RDM Login Msg. The Login Message encoder and decoder functions expect this type. It is a group of the classes of message that the Login domain supports. Any specific message class may be cast to an RsslRDMLoginMsg, and an RsslRDMLoginMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message. More...

RSSL_VA_API RsslRet rsslDecodeRDMLoginMsg ( RsslDecodeIterator *  pIter,
RsslMsg *  pMsg,
RsslRDMLoginMsg *  pLoginMsg,
RsslBuffer *  pMemoryBuffer,
RsslErrorInfo *  pError 
)
Parameters:
pEncodeIterThe Decode Iterator
pLoginMsgThe RDM Login Message to be populated
pMemoryBufferAn RsslBuffer pointing to space that may be used as additional storage space for the decoded message.
pErrorAn Error Structure, which will be populated if the decoding fails.
Returns:
RSSL_RET_SUCCESS, if the message was succesfully decoded and correctly followed the RDM.
RSSL_RET_FAILURE, if the message was not successfully decoded or did follow the RDM. Information about the error will be stored in pError.
RSSL_RET_BUFFER_TOO_SMALL, if the size of pNewMemoryBuffer was not large enough to store all additional data.
