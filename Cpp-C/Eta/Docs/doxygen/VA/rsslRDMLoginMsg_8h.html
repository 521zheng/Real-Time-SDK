union  RsslRDMLoginMsg
 The RDM Login Msg. The Login Message encoder and decoder functions expect this type. It is a group of the classes of message that the Login domain supports. Any specific message class may be cast to an RsslRDMLoginMsg, and an RsslRDMLoginMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message. More...
