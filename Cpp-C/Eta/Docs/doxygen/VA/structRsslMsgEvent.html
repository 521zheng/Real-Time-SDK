RsslMsgEvent Struct Reference
RSSL Reactor Channel Event Structure

Base event provided to Message Callback functions. This structure is provided to defaultMsgCallback functions and is present in structures provided to other Message Callback functions such as loginMsgCallback and directoryMsgCallback.
