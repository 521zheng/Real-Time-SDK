Enumeration Type Documentation

enum RsslGenericFlags
See also:
RsslGenericMsg, RsslMsg
Enumerator:
RSSL_GNMF_NONE 

(0x000) No RsslGenericFlags are present.

RSSL_GNMF_HAS_EXTENDED_HEADER 

(0x001) This RsslGenericMsg has an extended header buffer, contained in RsslGenericMsg::extendedHeader.

RSSL_GNMF_HAS_PERM_DATA 

(0x002) This RsslGenericMsg has permission information, contained in RsslGenericMsg::permData.

RSSL_GNMF_HAS_MSG_KEY 

(0x004) This RsslGenericMsg has a message key, contained in RsslGenericMsg::msgBase::msgKey.

RSSL_GNMF_HAS_SEQ_NUM 

(0x008) This RsslGenericMsg has a sequence number, contained in RsslGenericMsg::seqNum.

RSSL_GNMF_MESSAGE_COMPLETE 

(0x010) Indicates that this RsslGenericMsg is the final part of a multi-part generic message. This flag should be set on both single-part generic messages, as well as the final message in a multi-part generic message sequence.

RSSL_GNMF_HAS_SECONDARY_SEQ_NUM 

(0x020) This RsslGenericMsg has a secondary sequence number, contained in RsslGenericMsg::secondarySeqNum

RSSL_GNMF_HAS_PART_NUM 

(0x040) This RsslGenericMsg has a part number, contained in RsslGenericMsg::partNum.

RSSL_GNMF_HAS_REQ_MSG_KEY 

(0x080) The RsslGenericMsg has the original request's message key, contained in RsslGenericMsg::msgBase::reqMsgKey
