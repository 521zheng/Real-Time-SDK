Detailed Description

Details for encoding a specific primitive type can be found in that type's respective section.

Generic Primitive Type Encoding Example

The following example demonstrates generic encoding of an RsslReal.

Creation and initialization of the RsslEncodeIterator is not shown in the example, however see the example in Transport API Encode Iterator for more information. The example assumes that the encIter was properly cleared and has an appropriate size buffer and version information already set.

    //Create and populate the RsslReal with 654.23
    RsslReal rReal = {RSSL_FALSE, RSSL_RH_EXPONENT_2, 65423};
    RsslRet retVal;

    //Invoke rsslEncodePrimitiveType() function to encode content
    if ((retVal = rsslEncodePrimitiveType(&encIter, RSSL_DT_REAL, (void*)&rReal)) < RSSL_RET_SUCCESS)
    {
        printf("Error %s (%d) encountered with rsslEncodePrimitiveType.  Error Text: %s\n", 
                    rsslRetCodeToString(retVal), retVal, rsslRetCodeInfo(retVal));
    }

Functions

RSSL_API RsslRet rsslEncodePrimitiveType (RsslEncodeIterator *pIter, RsslPrimitiveType primitiveType, const void *pData)
 Used to encode any primitive type (e.g. RsslInt, RsslReal, RsslDate) into buffer referred to by RsslEncodeIterator. This function cannot be used for RsslArray encoding.
