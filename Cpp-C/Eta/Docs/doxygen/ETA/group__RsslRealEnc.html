RsslReal Encoding
RsslReal Reference Group

Detailed information about RsslReal encoding. More...

Detailed Description

When encoding an RsslReal, only the number of bytes necessary to represent the contained value will be encoded.

Note:
If sending RsslReal as blank, set the RsslReal::isBlank to RSSL_TRUE.

RsslReal Encoding Example

The following example demonstrates encoding an RsslReal.

Creation and initialization of the RsslEncodeIterator is not shown in the example, however see the example in Transport API Encode Iterator for more information. The example assumes that the encIter was properly cleared and has an appropriate size buffer and version information already set.

    //Create and populate the RsslReal with 654.23
    RsslReal rReal = {RSSL_FALSE, RSSL_RH_EXPONENT_2, 65423};
    RsslRet retVal;

    //Invoke rsslEncodeReal() function to encode content
    if ((retVal = rsslEncodeReal(&encIter, &rReal)) < RSSL_RET_SUCCESS)
    {
        printf("Error %s (%d) encountered with rsslEncodeReal.  Error Text: %s\n", 
                    rsslRetCodeToString(retVal), retVal, rsslRetCodeInfo(retVal));
    }
