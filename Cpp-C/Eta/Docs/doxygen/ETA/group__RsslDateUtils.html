RsslDate Utility & Conversion Functions
RsslDate Reference Group

Detailed information about RsslDate utility and conversion helper functions. More...

Detailed Description

There are several helper and conversion functions provided for use with the RsslDate. These can help with comparing and converting between RsslDate and various date strings.

Functions

 Verifies that day, month, and year information is valid (e.g. values that correspond to actual month or day information)
RSSL_API RsslRet rsslDateStringToDate (RsslDate *oDate, const RsslBuffer *iDateString)
 Converts string date in strftime()'s "%d %b %Y" format (e.g. 01 JUN 2003) or "%m/%d/%y" to RsslDate.
