Overview: Transport API Message Encode/Decode Interface Functions

This section contains a listing of the Transport API encode and decode functions available for the Transport API Messages.

RsslMsg Encoding Interface Function List

  • rsslEncodeMsg() - Encodes an RsslMsg from a fully populated message structure. If any message key attributes (RsslMsgKey::encAttrib), RsslMsgBase::extendedHeader, or payload (RsslMsgBase::encDataBody) are required, these must be set prior to this function call.
  • rsslEncodeMsgInit() - Encodes an RsslMsg, where header encoding will return control to the user so any message key attributes (RsslMsgKey::encAttrib), RsslMsgBase::extendedHeader, or payload (RsslMsgBase::encDataBody) content can be encoded. When encoding is complete, the user must call rsslEncodeMsgComplete().
  • rsslEncodeMsgKeyAttribComplete() - Used invoking rsslEncodeMsgInit(), if the application indicated that message key attributes (RsslMsgKey::encAttrib) should be encoded but did not provide them as pre-encoded. After rsslEncodeMsgInit() returns and the application has completed their attribute encoding, this function is called to complete attribute encoding in the message and allow for encoding any further payload or extended header content.
  • rsslEncodeExtendedHeaderComplete() - Used invoking rsslEncodeMsgInit(), if the application indicated that the extended header should be encoded but did not provide it as pre-encoded. After rsslEncodeMsgInit() (and rsslEncodeMsgKeyAttribComplete() if needing to encode attributes) returns and the application has completed their extended header encoding, this function is called to complete extended header encoding in the message and allow for encoding any further payload.
  • rsslEncodeMsgComplete() - Completes RsslMsg encoding. This is performed after completing all other portions of message encoding (e.g., after rsslEncodeMsgInit() completes and, if needed, rsslEncodeMsgKeyAttribComplete() and/or rsslEncodeExtendedHeaderComplete() complete.


RsslMsg Decoding Interface Function List

  • rsslDecodeMsg() - Decodes encoded message information into an RsslMsg. Any message key attribute (RsslMsgKey::encAttrib), extended header, and payload (RsslMsgBase::encDataBody) will not be decoded by this function call.
  • rsslDecodeMsgKeyAttrib() - If intending to decode any message key attribute (RsslMsgKey::encAttrib) using the same RsslDecodeIterator used when calling rsslDecodeMsg(), this function ensures the iterator is in the proper state to decode the attributes. This is necessary when using the same iterator because the attribute and the payload may have the same type, which introduces ambiguity in the users intent.
