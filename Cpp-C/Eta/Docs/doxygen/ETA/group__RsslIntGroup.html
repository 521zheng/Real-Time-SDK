RsslInt Reference Group
Transport API Data Primitive Types

The integer primitive type allows the user to represent signed integer content ranging from -263 to 263 - 1. More...

Detailed Description

The RsslInt functionality is defined in "rtr/rsslTypes.h", "rtr/rsslPrimitiveEncoders.h", and "rtr/rsslPrimitiveDecoders.h"

RsslInt Primitive Type Overview

The RsslInt primitive type allows the user to represent signed integers ranging from -263 to 263 - 1. This is defined as a type definition to the system's native 64-bit signed type.
See RsslInt Primitive Type for more detailed information.


RsslInt Encoding Functions

The following function is used while encoding an RsslInt.
See RsslInt Encoding for more detailed information.

  • rsslEncodeInt() - Encodes an RsslInt primitive type. This will perform any necessary byte swapping when encoding into the buffer referred to by the RsslEncodeIterator.



RsslInt Decoding Functions

The following function is used while decoding an RsslInt.
See RsslInt Decoding for more detailed information.

  • rsslDecodeInt() - Decodes an RsslInt primitive type. This will perform any necessary byte swapping when decoding from the buffer referred to by the RsslDecodeIterator.
