Generic Primitive Type Decoding
Generic Primitive Type Utilities

Detailed information about generic primitive type decoding functions. More...

Detailed Description

Details for decoding a specific primitive type can be found in that type's respective section.

Note:
When decoding a blank value, RSSL_RET_BLANK_DATA is returned and the value is populated based on the blanking rules defined in each type's respective section.

Generic Primitive Type Decoding Example

The following example demonstrates generic decoding. This assumes the type was already known or looked up in a field dictionary. 

    RsslInt rInt; //No need to initialize since we will decode into this
    RsslRet retVal;

    retVal = rsslDecodePrimitiveType(&decIter, RSSL_DT_INT, &rInt);
    {
        //handle error as needed
        switch(retVal)
        {
            case RSSL_RET_BLANK_DATA:
                //Decoded content was blank
            break;
            case RSSL_RET_SUCCESS:
                //Decoding was successful, do something with content
                ...
            break;
            default:
                printf("Error %s (%d) encountered with rsslDecodePrimitiveType.  Error text: %s\n", 
                    rsslRetCodeToString(retVal), retVal, rsslRetCodeInfo(retVal));
        }
    }

Functions

RSSL_API RsslRet rsslDecodePrimitiveType (RsslDecodeIterator *pIter, RsslPrimitiveType primitiveType, void *oValue)
 Used to decode any primitive type (e.g. RsslInt, RsslReal, RsslDate) from buffer referred to by RsslDecodeIterator. This function cannot be used for RsslArray decoding.
