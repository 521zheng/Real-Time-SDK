RsslDataDictionary Struct Reference
Data Dictionary

A data dictionary Houses all known fields loaded from a field dictionary and their corresponding enum types loaded from an enum type dictionary. The dictionary also saves general information about the dictionary itself -- this is found in the "!tag" comments in the file or in the summary data of dictionaries encoded via the official domain model.
