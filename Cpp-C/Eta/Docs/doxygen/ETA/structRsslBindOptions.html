 The size (in kilobytes) of the system's receive buffer used for this connection, where applicable. Setting of 0 indicates to use default sizes. This setting will carry through on all connections accepted with rsslAccept(). RsslUInt32 sysSendBufSize
 The size (in kilobytes) of the system's send buffer used for this connection, where applicable. Setting of 0 indicates to use default sizes. This setting will carry through on all connections accepted with rsslAccept(). This can also be set or changed per channel via rsslIoctl(), however setting prior to connection may allow for larger sizes to be specified.
