RsslDateTime Utility & Conversion Functions
RsslDateTime Reference Group

Detailed information about RsslDateTime utility and conversion helper functions. More...

Detailed Description

There are several helper and conversion functions provided for use with the RsslDateTime. These can help with comparing and converting between RsslDateTime and various date and time strings.

Functions

 Verifies that day, month, year, hour, minute, second, and millisecond information is valid (e.g. values that correspond to actual date and time information)
RSSL_API RsslRet rsslDateTimeLocalTime (RsslDateTime *oDateTime)
 Set the given RsslDateTime to now/current time and date in the local time zone.
RSSL_API RsslRet rsslDateTimeStringToDateTime (RsslDateTime *oDateTime, const RsslBuffer *iDateTimeString)
 Converts DateTime string to RsslDateTime, expects date before time, where date is formatted in in strftime()'s "%d %b %Y" format (e.g. 01 JUN 2003) or "%m/%d/%y" and time is formatted in strftime()'s "%H:%M" or "%H:%M:%S" format (e.g. 13:01 or 15:23:54).
RSSL_API RsslRet rsslDateTimeToString (RsslBuffer *oBuffer, RsslUInt8 dataType, RsslDateTime *iDateTime)
