RsslFloat Reference Group
Transport API Data Primitive Types

The float primitive type allows the user to represent 4-byte IEEE-754 floating point values. More...

Detailed Description

The RsslFloat functionality is defined in "rtr/rsslTypes.h", "rtr/rsslPrimitiveEncoders.h", and "rtr/rsslPrimitiveDecoders.h"

RsslFloat Primitive Type Overview

The RsslFloat primitive type allows the user to represent IEEE-754 floating point values. This is defined as a type definition to the system's native float type.
See RsslFloat Primitive Type for more detailed information.


RsslFloat Encoding Functions

The following function is used while encoding an RsslFloat.
See RsslFloat Encoding for more detailed information.

rsslEncodeFloat() - Encodes an RsslFloat primitive type. This will perform any necessary byte swapping when encoding into the buffer referred to by the RsslEncodeIterator.


RsslFloat Decoding Functions

The following function is used while decoding an RsslFloat.
See RsslFloat Decoding for more detailed information.

  • rsslDecodeFloat() - Decodes an RsslFloat primitive type. This will perform any necessary byte swapping when decoding from the buffer referred to by the RsslDecodeIterator.
