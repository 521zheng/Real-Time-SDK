RSSL_PSMF_NONE 

(0x000) No RsslPostMsg flags are present

RSSL_PSMF_HAS_EXTENDED_HEADER 

(0x001) This RsslPostMsg has an extended header buffer, contained in RsslPostMsg::extendedHeader.

RSSL_PSMF_HAS_POST_ID 

(0x002) This RsslPostMsg has a Post ID, contained in RsslPostMsg::postId.

RSSL_PSMF_HAS_MSG_KEY 

(0x004) This RsslPostMsg has a message key, contained in RsslPostMsg::msgBase::msgKey.

RSSL_PSMF_HAS_SEQ_NUM 

(0x008) This RsslPostMsg has a sequence number, contained in RsslPostMsg::

RSSL_PSMF_POST_COMPLETE 

(0x020) Indicates that this RsslPostMsg is the final part of the post message. This flag should also be set on single-part post messages, as well as the final message in a multi-part RsslPostMsg sequence.

RSSL_PSMF_ACK 

(0x040) Indicates that the provider receiving this RsslPostMsg should send an RsslAckMsg to indicate that this RsslPostMsg has been received and processed. The provider must include the RsslPostMsg::postId in the RsslAckMsg's RsslAckMsg::ackId.

RSSL_PSMF_HAS_PERM_DATA 

(0x080) This RsslPostMsg has permission authorization data, contained in RsslPostMsg::permData.

RSSL_PSMF_HAS_PART_NUM 

(0x100) This RsslPostMsg has a part number, contained in RsslPostMsg::partNum.

RSSL_PSMF_HAS_POST_USER_RIGHTS 

(0x200) This RsslPostMsg has post user rights, contained in RsslPostMsg::postUserRights.
