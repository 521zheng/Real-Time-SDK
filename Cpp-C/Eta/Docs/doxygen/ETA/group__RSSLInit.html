Enumeration Type Documentation

enum RsslLockingTypes
See also:
rsslInitialize
Enumerator:
RSSL_LOCK_NONE 

(0) No locking will be enabled. Use if only one thread is using RSSL.

RSSL_LOCK_GLOBAL_AND_CHANNEL 

(1) Global lock and per-channel locks enabled. Can access functions from different threads

RSSL_LOCK_GLOBAL 

(2) Only the global lock is enabled. RsslChannels can be used by different threads, however each channel should not be used by more than one thread.
