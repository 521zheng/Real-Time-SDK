RsslInt Encoding
RsslInt Reference Group

Detailed information about RsslInt encoding. More...

Detailed Description

When encoding an RsslInt, only the number of bytes necessary to represent the contained value will be encoded.

RsslInt Encoding Example

The following example demonstrates encoding an RsslInt.

Creation and initialization of the RsslEncodeIterator is not shown in the example, however see the example in Transport API Encode Iterator for more information. The example assumes that the encIter was properly cleared and has an appropriate size buffer and version information already set.

    //Create and populate the RsslInt
    RsslInt rInt = -123456;
    RsslRet retVal;

    //Invoke rsslEncodeInt() function to encode content
    if ((retVal = rsslEncodeInt(&encIter, &rInt)) < RSSL_RET_SUCCESS)
    {
        printf("Error %s (%d) encountered with rsslEncodeInt.  Error Text: %s\n", 
                    rsslRetCodeToString(retVal), retVal, rsslRetCodeInfo(retVal));
    }
