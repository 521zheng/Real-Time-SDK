enum RsslAckFlags

RsslAckMsg, RsslMsg
Enumerator:
RSSL_AKMF_NONE 

(0x00) No Flags have been set

RSSL_AKMF_HAS_EXTENDED_HEADER 

(0x01) Indicates that this RsslAckMsg has an extended header buffer, contained in RsslAckMsg::extendedHeader.

RSSL_AKMF_HAS_TEXT 

(0x02) Indicates that this RsslAckMsg has a text buffer, contained in RsslAckMsg::text.

RSSL_AKMF_PRIVATE_STREAM 

(0x04) Indicates that this RsslAckMsg acknowledges the establishment of a private stream

RSSL_AKMF_HAS_SEQ_NUM 

(0x08) Indicates that this RsslAckMsg has a sequence number, contained in RsslAckMsg::seqNum.

RSSL_AKMF_HAS_MSG_KEY 

(0x10) Indicates that this RsslAckMsg has a Message Key, contained in RsslAckMsg::msgBase::msgKey.

RSSL_AKMF_HAS_NAK_CODE 

(0x20) Indicates that this RsslAckMsg has a NAK Code, contained in RsslAckMsg::nakCode. See RsslNakCodes.

RSSL_AKMF_QUALIFIED_STREAM 

(0x40) Indicates that this RsslAckMsg acknowledges the establishment of a qualified stream request.

enum RsslNakCodes
See also:
RsslAckMsg, RsslMsg
Enumerator:
RSSL_NAKC_NONE 

(0) No Nak Code

RSSL_NAKC_ACCESS_DENIED 

(1) Access Denied. The user not properly permissioned for posting on the item or service.

RSSL_NAKC_DENIED_BY_SRC 

(2) Denied by source. The source being posted to has denied accepting this post message.

RSSL_NAKC_SOURCE_DOWN 

(3) Source Down. Source being posted to is down or not available.

RSSL_NAKC_SOURCE_UNKNOWN 

(4) Source Unknown. The source being posted to is unknown and is unreachable.

RSSL_NAKC_NO_RESOURCES 

(5) No Resources. A component along the path of the post message does not have appropriate resources available to continue processing the post.

RSSL_NAKC_NO_RESPONSE 

(6) No Response. This code may mean that the source is unavailable or there is a delay in processing the posted information.

RSSL_NAKC_GATEWAY_DOWN 

(7) Gateway is Down. The gateway device for handling posted or contributed information is down or not available. (8) Reserved (9) Reserved

RSSL_NAKC_SYMBOL_UNKNOWN 

(10) Unknown Symbol. The item information provided within the post message is not recognized by the system.

RSSL_NAKC_NOT_OPEN 

(11) Item not open. The item being posted to does not have an available stream open.

RSSL_NAKC_INVALID_CONTENT 

(12) Nak being sent due to invalid content. The content of the post message is invalid and cannot be posted, it does not match the expected formatting for this post.
