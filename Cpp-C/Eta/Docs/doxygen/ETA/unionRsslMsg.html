RsslMsg Union Reference
RsslMsg Union Reference Group

The OMM Message Union. All message encoding and decoding functions expect the RsslMsg type. Any specific message class can be cast to the RsslMsg, and an RsslMsg can be cast to any specific message class. When decoding, the RsslMsg.msgBase contains common members that can be used to identify the specific message class or domain type. More...
